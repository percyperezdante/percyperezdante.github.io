Projects

I am and was involved in the following projects.

Current projects

  • Java based Named Data Networkig Forwarder Deamon JNFD

    This space presents Java NDN Forwarder Daemon (JNFD), which is a prototype implementation of nMANETs and a proof-of-concept tool. The design and implementation of JNFD was from the beginning though to be deployed in Android mobile devices an Linux based terminals. JNFD is tool for experimentation and not for production yet. JNFD intends to help in the exploration and evidence generation of how NDN protocol can solve the limitations that MANET are still facing.

  • Mini-JNFD

    Mini-JNFD is the application of JNFD in Mininet-Wifi in order to test, detect and simulate how JNFD nodes may behave when they interact in an Ad-hoc environment. Mini-JNFD uses two types of mobility: mobility models and read GPS traces. Mini-JNFD is reproducible and scalable, it can be easy deployed in a local Virtual Box through Vagrant and available experiments can be re-executed using the tools provided by Mini-JNFD.

Past projects

  • Land line telephony provisioning system CORHAL

    CORHAL was designed to automated the provisioning of telephone landlines in SIEMENS and ALCATEL switching newtorks for local and long distance communication. It executes provisioning requests in the switching network and replies the status of the execution to the requester automatically. CORHAL can provide more than 20 types of provisioning services, it starts to be operative in 2004 and it seems active until now. Sadly the code is not open source yet.

  • Billing data acquisition FTAM

    FTAM manages billing files between SIEMENS switching network and local servers who collect, prepare and deliver them to dedicated internal customers. FTAM helps to deliver fresh billing files on time in order to achieve finance goals during each month and contribute to reduce delay to deliver billing information to final customer. FTAM was designed exclusively for local and long distance SIEMENS and ALCATEL switching networks of the whole company.
